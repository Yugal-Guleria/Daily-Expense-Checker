# 💸 Daily Expense Checker

A simple Python application that tracks daily expenses, compares them with a set budget, and gives clear spending insights with useful advice.

---

## 🚀 Features

* Tracks daily expenses (Food, Transport, Other)
* Lets you set and compare with a daily budget
* Shows spending status based on usage
* Provides category-wise expense analysis
* Gives simple financial advice
* Clean and easy-to-read output

---

## 📊 Expense Categories

| Category  | Description              |
| --------- | ------------------------ |
| Food      | Meals, snacks, groceries |
| Transport | Travel, fuel, tickets    |
| Other     | Miscellaneous expenses   |

---

## 📈 Budget Status Logic

| Spending Ratio (Total / Budget) | Status         | Meaning                      |
| ------------------------------- | -------------- | ---------------------------- |
| ≤ 50%                           | Excellent ✅    | Strong control over spending |
| 50% – 80%                       | Healthy 🟢     | Balanced spending            |
| 80% – 100%                      | Caution ⚠️     | Close to budget limit        |
| > 100%                          | Over Budget 🚨 | Budget exceeded              |

---

## ⚠️ Smart Alerts

* Food > 40% → High food spending
* Transport > 30% → Travel cost is high
* Other > 20% → Unnecessary expenses detected
